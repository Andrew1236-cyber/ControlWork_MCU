## Описание проекта
Система сбора данных с микроконтроллера через Ethernet интерфейс с последующей синхронизацией в RabbitMQ.

## Архитектура системы
Микроконтроллер → Ethernet → Приемник (Python/Flask) → SQLite → Синхронизатор → RabbitMQ

## Задание №1: Сбор данных в SQLite ✅ ВЫПОЛНЕНО
- Разработана прошивка микроконтроллера с Ethernet
- Реализован прием данных по HTTP на порту 8080
- Данные сохраняются в SQLite базу
- Сохраняется device_id, значения и временная метка

## Задание №2: Синхронизация с RabbitMQ ✅ ВЫПОЛНЕНО
- Реализована синхронизация данных из SQLite в RabbitMQ
- После успешной отправки данные удаляются из SQLite
- Обеспечена надежная доставка сообщений
